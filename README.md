# 🔐 Secret Language (Python)

This is a beginner-friendly Python project that encodes and decodes text using a custom secret language algorithm. The program transforms a normal message into a secret format and can decode it back to its original form.

## Concepts Used

* Python 3
* `random` Module
* User Input (`input()`)
* Conditional Statements (`if`, `elif`, `else`)
* Loops (`for`)
* String Slicing
* String Concatenation
* Variables
* Basic Input Validation
* Custom Encoding & Decoding Logic

## How It Works

* Takes a message from the user.
* Generates random characters using the `random` module.
* Rearranges the message using string slicing.
* Adds random characters at the beginning and end of the message during encoding.
* Removes the added characters and restores the original message during decoding.

## Purpose

This project was created to practice Python fundamentals such as string manipulation, loops, conditional statements, and logical problem-solving while building a fun text encoder and decoder.
